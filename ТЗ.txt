🚀 Фазы развития
Фаза 1 — Улучшение UX
Сложность: ⭐ Легко | Приоритет: 🔴 Высокий
1. Пустое состояние (Empty State)
   ├── Картинка-заглушка если карточек нет
   ├── Текст "Карточек пока нет"
   └── Кнопка "Добавить первую карточку"

2. Валидация URL картинки
   ├── Проверка формата URL
   ├── Проверка расширения (jpg, png, webp...)
   └── Подсветка ошибки в поле ввода

3. Обработка ошибки загрузки картинки в карточке
   ├── Если картинка не загрузилась
   └── Показывать placeholder изображение

4. Анимации карточек
   ├── Появление при загрузке (Grow / Fade)
   ├── Исчезновение при удалении
   └── Подсветка при добавлении новой карточки
Фаза 2 — Поиск и фильтрация
Сложность: ⭐⭐ Средне | Приоритет: 🔴 Высокий
1. Поиск по названию
   ├── Строка поиска в Header
   ├── Фильтрация в реальном времени
   └── Подсветка найденного текста

2. Сортировка карточек
   ├── По дате добавления (новые / старые)
   ├── По названию (А-Я / Я-А)
   └── Выпадающий список сортировки

3. Фильтрация по категориям
   ├── Добавить поле "категория" в карточку
   ├── Чипы для фильтрации
   └── Множественный выбор категорий

4. Счётчик результатов
   └── "Найдено: 5 карточек"
Фаза 3 — Авторизация
Сложность: ⭐⭐⭐ Сложно | Приоритет: 🟡 Средний
1. Регистрация / Вход
   ├── Форма регистрации (имя, email, пароль)
   ├── Форма входа (email, пароль)
   └── JWT токены

2. Защита маршрутов
   ├── Неавторизованный видит только карточки
   ├── Авторизованный может управлять карточками
   └── React Router — приватные маршруты

3. Профиль пользователя
   ├── Аватар
   ├── Имя пользователя в Header
   └── Кнопка выхода

4. Привязка карточек к пользователю
   ├── Каждая карточка принадлежит автору
   ├── Редактировать/удалять только свои
   └── Показывать автора на карточке
Фаза 4 — Лайки и взаимодействие
Сложность: ⭐⭐ Средне | Приоритет: 🟡 Средний
1. Лайки на карточках
   ├── Кнопка лайка с счётчиком
   ├── Анимация при лайке
   └── PUT /api/cards/:id/likes

2. Избранное
   ├── Добавить в избранное
   ├── Фильтр "Показать избранное"
   └── Хранение в localStorage

3. Просмотр карточки
   ├── Модальное окно с полной информацией
   ├── Увеличенное изображение
   └── Все детали карточки
Фаза 5 — Технические улучшения
Сложность: ⭐⭐⭐ Сложно | Приоритет: 🟢 Низкий
1. React Router
   ├── /cards — список карточек
   ├── /cards/:id — детальная страница
   ├── /profile — профиль пользователя
   └── /login, /register — авторизация

2. Оптимистичные обновления
   ├── UI обновляется сразу
   ├── Запрос идёт в фоне
   └── Откат при ошибке

3. Пагинация / Бесконечная прокрутка
   ├── GET /api/cards?page=1&limit=12
   ├── Кнопка "Загрузить ещё"
   └── или Intersection Observer

4. Кэширование запросов
   ├── Установить React Query / SWR
   ├── Автоматическое обновление данных
   └── Повторные запросы при ошибке

5. TypeScript
   ├── Типизация props компонентов
   ├── Типизация API ответов
   └── Интерфейсы для карточек
Фаза 6 — Инфраструктура
Сложность: ⭐⭐⭐ Сложно | Приоритет: 🟢 Низкий
1. Деплой
   ├── Frontend → Vercel / Netlify
   ├── Backend → Railway / Render
   └── MongoDB → MongoDB Atlas

2. Переменные окружения
   ├── .env для dev и prod
   └── Vite import.meta.env

3. Docker
   ├── Dockerfile для frontend
   ├── Dockerfile для backend
   └── docker-compose.yml

4. Тесты
   ├── Unit тесты хуков (Vitest)
   ├── Тесты компонентов (Testing Library)
   └── E2E тесты (Playwright)
